# Evolutionary Simulation

An evolutionary simulation where entities with heritable traits compete in a dynamic environment.

## Overview

This simulation demonstrates evolutionary principles through autonomous entities that:
- Possess **heritable traits** that pass to offspring
- Undergo **hue mutation** to create evolving lineages (visualized through colors)
- **Compete for survival** in a dynamic environment

## Elements

| Element | Meaning |
|---------|---------|
| **Entity Colors** | Evolving lineages (Hue mutation) |
| **Green Background** | Food (beneficial) |
| **Purple Background** | Toxin (harmful) |

## Features

- **Population Tracking** - Real-time population count
- **Time Tracking** - Simulation elapsed time
- **Dynamic Environment** - Entities adapt to changing conditions

## Usage

1. Open `app.html` in any modern web browser
2. Watch the simulation run automatically
3. Observe how different colored lineages evolve and compete

## Technologies

- HTML5
- JavaScript
- Canvas API
